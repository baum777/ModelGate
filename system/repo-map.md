- `server/`: Fastify backend authority layer
- `web/`: Vite/React console shell

- `README.md`: repo front door

- `docs/test-matrix.md`: verification surface
- `docs/ui-spec.md`: UI intent
- `docs/vercel-deployment.md`: deployment notes

## Non-Authoritative Or Derived Areas

- `output/`: generated or collected output
- `artifacts/`: evidence artifacts
